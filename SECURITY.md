# Security Policy

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | ✅ 安全更新 |

## 报告安全漏洞

如果你发现了安全漏洞，请**不要**通过公开渠道报告。

请通过以下方式私密报告：

1. 发送邮件至项目维护者：security@ricequant.com

### 报告内容

请包含以下信息：

- 漏洞描述
- 复现步骤
- 影响范围
- 建议的修复方式（如果有）

### 响应时间

- 确认收到：48 小时内
- 初步评估：7 天内
- 修复发布：视严重程度，通常 30 天内

## 安全最佳实践

使用 RQData CLI 时：

- 使用环境变量存储凭据，避免在命令行中明文传递
- 定期更新到最新版本
- 不要将 `~/.rqdata/` 目录提交到版本控制
