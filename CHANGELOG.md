# Changelog

## [1.0.0] - 2026-03-27

### 首次发布

**核心特性**
- Go 实现的高性能 CLI 工具
- 支持 68 个金融数据 API 命令
- NDJSON/JSON/CSV 多格式输出
- 系统 Keyring 集成，安全存储凭证
- 自动 Token 管理和刷新
- Schema 自动生成，含中文参数描述

**数据覆盖**

- 期货数据（8 个命令）
- 期权数据（7 个命令）
- 宏观数据（6 个命令）

**技术亮点**
- 单文件可执行，静态编译
- 跨平台支持（Linux/macOS/Windows）
- 类型安全的参数验证
- AI Agent 友好的 NDJSON 输出

**文档**
